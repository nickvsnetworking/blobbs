# Blobbs — Tadpole Pond

A relaxed, tap-to-grow pond game. Tap a tadpole to grow it into a tadlet, tap
again for a frog, then one more tap and it hops off the screen. Nurture the
whole pond!

- **Tadpoles are born tiny** and grow in, getting larger the longer they swim.
- **Tap to advance a stage**: tadpole → tadlet → frog → hop away.
- Frogs leap off with a real crouch-and-spring jump.
- No timer, no failure — just a calm pond to tend.

## Play

It's a single self-contained HTML file — no build step, no dependencies.

Open `index.html` in any modern browser, or play the deployed version.

## Tech

Vanilla HTML/CSS/JS with a `<canvas>` render loop and procedural WebAudio sound.
All sprites are drawn procedurally (no image assets).
